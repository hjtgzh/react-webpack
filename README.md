## 项目简介

react、webpack 从零开始搭建

## 项目技术栈

react@16.13.1 + react-router-dom@5.2.0 + webpack@4.43.0 + less@3.11.3 + antd@4.4.1

## 项目运行

```
git clone --depth 1 https://github.com/hjtgzh/react-webpack.git

cd react (进入项目)

npm install (安装依赖包)

npm start (启动服务)

```

### 建议使用 npm 下载，不推荐 cnpm

## 项目构建

```
npm run build (正式环境的打包部署)

```
